# 🏪 City Smart Retail — Excel Analytics Dashboard

A end-to-end data analytics project built entirely in Microsoft Excel,
simulating a real retail chain operating across 5 cities in Maharashtra.

## 📊 Dashboard Sheets

| Sheet | Description |
|---|---|
| 🏠 Home | Navigation hub |
| 📊 Executive Summary | KPI cards, dynamic headline, sparklines |
| 📈 Sales Analysis | Monthly trends, combo chart, slicers |
| 👥 Customer Insights | RFM scoring, loyalty breakdown, SUMIFS |
| 📦 Inventory Tracker | Stock alerts, Data Validation, status flags |
| 💰 Profit & Loss | Live P&L, Goal Seek demo, Waterfall chart |
| 🏆 Store Comparison | Weighted scoring, Radar chart, Head-to-Head |

## 🛠️ Excel Skills Demonstrated

- **Formulas:** XLOOKUP, INDEX MATCH, SUMIFS, COUNTIFS,
  AVERAGEIF, IFERROR, IF/IFS, RANK, TEXT, EDATE
- **Data Tools:** Pivot Tables, Power Query, Slicers,
  Data Validation, Goal Seek
- **Visualization:** Combo Chart, Waterfall Chart, Radar Chart,
  Doughnut Chart, Sparklines, Conditional Formatting
- **UX:** Navigation buttons, Sheet protection,
  Dynamic text cells, Color coded tabs

## 📁 Dataset

- 500 rows of transactional sales data
- Period: January 2024 — December 2024
- 5 stores: Mumbai, Pune, Nagpur, Nashik, Aurangabad
- 4 categories: Electronics, Clothing, Groceries, Home & Kitchen
- 200 unique customers with RFM scoring

## 🚀 How to Use

1. Download `City_Smart_Retail.xlsx`
2. Open in Microsoft Excel (2016 or later)
3. Start from the HOME sheet
4. Use slicers to filter by Category or Store
5. Try Goal Seek on the P&L sheet — change COGS % to hit target profit
